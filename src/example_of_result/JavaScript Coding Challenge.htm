About

This website is a JavaScript Coding Challenge to do the following tasks:

  1. Provide a text input field which takes the query param.
  2. Wait for 200ms if any change in the search field.
  3. Reset the time counter and start it if there any change in that timeframe.
  4. Fire off an Ajax call to load the data.
  5. If the input changes again during the runtime of the Ajax call, cancel it.
  6. Display a list or grid of products with their images and names.
